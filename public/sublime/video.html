SublimeVideo

HTML5 Video Player

Jilion Twitter

Dartmoor by James Watson

Current features (in this demo)

  • HTML5 video goodness:
    • no browser plugin, no Flash dependencies
    • jump anywhere in the video and it'll start buffering from that point
  • Custom styled controls
  • Full-window mode:
    • Sleek zoom-in/out transitions
    • Live-resizing (when resizing the browser window)
    • More advanced controls on a draggable pane
    • Press spacebar to play/pause video
  • Full-screen mode by alt-clicking on the full-window button (currently only supported in latest WebKit Nightly Builds)

Features we plan to add (in the public release)

  • Firefox support
  • Volume controls
  • IE support by falling back to Flash

Supported Browsers

  • Safari (v4.0.4+)
  • Google Chrome (v4.0+)
  • Internet Explorer with Chrome Frame installed
